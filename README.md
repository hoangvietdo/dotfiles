## dotfiles

My dotfiles for 
- tmux (.tmux.conf)
- vim (.vimrc) - deprecated
- neovim (init.lua)
- oh-my-zsh (.zshrc)
- neofetch (config.conf)

for MacOS (Monterey) and Ubuntu (20.04).

## Usages
Run the auto-install shell script `./install.sh`

For Ubuntu, log out and log in again or reboot after installation.
